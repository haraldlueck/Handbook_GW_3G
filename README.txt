This directory contains the LaTeX guide and template for chapters to the Handbook of Gravitational Wave Astronomy:
 

1. author.tex   : the template file. If latex is used from command line, compile with sequence latex, bibtex, latex, latex.

2. author.pdf   : a compiled version of author.tex

3. svmult.cls   : the latex class file
4. reference.tex : List of references as examples
5. figure.eps   : Sample figure used
6. refguide.pdf : Reference guide for the SVMult Document Class
